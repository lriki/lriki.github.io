Detailed Description

スプライトオブジェクトを表します。

デフォルトの BlendMode は Alpha です。 デフォルトの CullMode は None です。（両面表示となります）

void ln::Sprite::setAnchorPoint ( const Vector2 &  value)

スプライトの原点を設定します。

値は (0,0) から (1,1) の間で指定します。 デフォルトは (0,0) で、これはスプライトの左下が原点であることを意味します。 (0.5,0.5) はスプライトの中央、(1,1) は右上が原点となります。アンカーポイント (原点) を設定します。デフォルトは中心(0.5, 0.5) です。
